# 优雅标签页管家 (Tab Manager Extension)

一个轻量、现代化的 Chrome 浏览器插件（基于最新的 Manifest V3 开发）。它可以帮助你一键导出当前窗口的所有标签页，或者按顺序延迟导入并打开这些标签页。

## ✨ 功能特点

* **一键导出**：将当前浏览器窗口中所有标签页的 URL 和标题，按从左到右的顺序导出为 `.txt` 文件。
* **平滑导入**：读取指定的 `.txt` 文件，并以 **0.2 秒的延迟** 逐个打开网页，防止瞬间打开大量标签页导致浏览器卡顿或请求被拦截。
* **现代 UI**：采用符合现代审美的极简设计，包含优雅的渐变色和阴影效果。
* **后台稳定运行**：导入任务交由后台 Service Worker 处理，即使关闭了插件弹窗，打开网页的任务也不会中断。

## 🚀 安装指南

由于此插件尚未发布到 Chrome 网上应用店，你可以通过“开发者模式”在本地安装使用：

1. 下载或 Clone 本仓库代码到本地文件夹。
2. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
3. 开启页面右上角的 **“开发者模式”** 开关。
4. 点击左上角的 **“加载已解压的扩展程序”**。
5. 选择本项目的文件夹，即可完成安装。

## 💡 使用方法

1. 点击浏览器工具栏中的插件图标打开面板。
2. **导出**：点击“导出当前标签页”，浏览器会提示你保存一个 `tabs_export.txt` 文件。
3. **导入**：点击“导入标签页文件”，选择符合上述格式的 `.txt` 文件，插件将会在后台排队依次为你打开这些网页。

## 📂 导入/导出格式说明

插件使用的文本格式非常简单，每一行代表一个标签页，URL 和标题之间使用 `,,,` 分隔：

```text
[https://www.google.com](https://www.google.com),,,Google
[https://github.com](https://github.com),,,GitHub
```
